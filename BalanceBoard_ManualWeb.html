Set-Up

Download and install Arduino

Run the RGBLamp.ino firmware with Arduino and upload the program on the Arduino UNO

You’ll need to install the Adafruit Neopixel library: Sketch/Include a library/Manage libraries

Search for «Adafruit Neopixel» in the search tab and the install latest version

Upload the program on your board

BalanceBoard/02_ArduinoFirmware/RGBLamp/

Download and install Processing

Run the RGBLamp.pde file with Processing while the lamp is plugged on a USB port of your computer

You’ll need to install the oscP5 library: Sketch/Import library.../Add a library...

Search for «oscP5» in the search tab and the install latest version

Go to line 40 into the Processing code and set the USB port where your Arduino is plugged

BalanceBoard/03_ProcessingApplication/RGBLamp/

Download and install Movuina

Launch Movuina and set-up by following the Quick Start tutorial

Once its done, you can play with the XMM algorithm developped by IRCAM

The data are automatically sent to Processing and you can enjoy the reaction!
